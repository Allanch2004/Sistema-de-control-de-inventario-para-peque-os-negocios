# StockControl 

Sistema de gestión de inventario desarrollado en lenguaje C como Proyecto Integrador.

## Descripción

StockControl C es una aplicación de consola diseñada para ayudar a pequeños negocios a administrar su inventario de manera organizada. El sistema permite registrar productos, controlar existencias, registrar ventas y generar reportes, manteniendo toda la información almacenada mediante archivos para garantizar su disponibilidad entre diferentes ejecuciones.

## Objetivo

Desarrollar una solución informática que facilite la administración del inventario de pequeños negocios, permitiendo controlar productos, existencias y ventas mediante una aplicación desarrollada completamente en lenguaje C.

## Funcionalidades

- Registro de productos.
- Visualización de productos registrados.
- Búsqueda de productos por código o nombre.
- Modificación de productos.
- Eliminación de productos.
- Registro de ventas.
- Actualización automática del inventario.
- Visualización de productos con bajo stock.
- Generación de reportes.
- Almacenamiento permanente de la información mediante archivos.

## Tecnologías utilizadas

- Lenguaje C
- Programación estructurada
- Estructuras (`struct`)
- Funciones modularizadas
- Arreglos
- Manejo de archivos
- Algoritmos de búsqueda y ordenamiento
- Memoria dinámica

## Archivos utilizados

- `productos.dat`
- `ventas.dat`

## Requisitos

- Compilador compatible con lenguaje C (GCC, Dev-C++, Code::Blocks, Visual Studio, entre otros).

## Ejecución

1. Compilar el proyecto.
2. Ejecutar el archivo generado.
3. Utilizar el menú principal para administrar el inventario.

## Menú principal

1. Registrar producto
2. Mostrar productos
3. Buscar producto
4. Modificar producto
5. Eliminar producto
6. Registrar venta
7. Actualizar inventario
8. Mostrar productos con bajo stock
9. Generar reportes
10. Guardar información
11. Salir
